We released our first VB.NET application!

Forced Crash is an application that can force a VB.NET app to crash instantly and show a pop-up message. The project file was created in Visual Studio 2010 and VB.NET. Our webpage that our software was released was created in HTML, CSS and JS.

To download the source code, run the following command:
git clone https://github.com/JoseErnestoOnGithub/GogleFoundation-ForceCrash-VB.NET.git

Then, open the project file that you have downloaded in Visual Studio. Now, build the project by clicking the green play button in the top of the Visual Studio window, or by pressing F5 on the computer keyboard. The project will compile successfully. See what happens by forcing a crash by clicking the Crash button. It will generate an error message when the button is clicked. When the OK button is clicked on the message, the app will instantly close. The exception details will be printed in the output.
